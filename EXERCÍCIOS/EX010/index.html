2 tags para as abertura do link

  1. Tag "_blank" - vai fazer o hiperlink abrir em uma nova janela: Nova janela, clique aqui!
  2. Tag "_self" - vai abrir o link na janela ou frame atual (modo padrão): Abrir na janela ou frame atual, clique aqui!

Link meu ou não?

Tem uma diferença entre usar um link para abrir um site seu e/ou de outras pessoas, para isso temos um tag que vai melhorar a natureza do destino, tag: rel

4 valores para a tag: rel

  1. Tag: next - usada para próxima parte do doc atual. - clique aqui!
  2. Tag: prev - usada para voltar a parte anterior do doc atual. - clique aqui!
  3. Tag: author - usada pra ir ao site do autor do artigo autal citado. - clique aqui!
  4. Tag: external - usada para ir ao site de outro, que não lhe envolve em nada e nem faz parte do seu site. - clique aqui!
  5. Tag: nofollow - usada para ir a um site não endossado, como um link pago. - clique aqui!
